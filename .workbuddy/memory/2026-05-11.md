## 修复复习模块正确答案显示 undefined

### 根因
- `ReviewController::list()` 查询时 select 字段缺少 `correct_answer`
- `ReviewPanel.js` 前端使用 `q.answer` 而非 `q.correct_answer` 来判断对错和显示正确答案
- 数据库字段名是 `correct_answer`，API 未返回该字段 → 前端 `q.answer` / `q.correct_answer` 均为 undefined

### 修复
1. `ReviewController.php` 第45行：select 列增加 `correct_answer`
2. `ReviewPanel.js` 第110行：`q.answer` → `q.correct_answer`（判断对错）
3. `ReviewPanel.js` 第125行：`q.answer` → `q.correct_answer`（显示正确答案）
